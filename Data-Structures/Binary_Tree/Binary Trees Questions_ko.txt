섹션 E - 이진 트리
[이 섹션에서 지정된 문제 1개를 답안으로 제출]

안내:
문제별 프로그램 템플릿은 APAS 시스템에 제공됩니다.
함수를 구현할 때 반드시 해당 템플릿을 사용해야 합니다.

1. (identical)
두 개의 이진 트리 `tree1`, `tree2`가 구조적으로 동일한지 판단하는 재귀 C 함수 `identical()`을 작성하시오.
두 트리가 모두 비어 있으면 동일합니다.
두 트리가 모두 비어 있지 않고, 같은 값을 가지며, 왼쪽과 오른쪽 서브트리도 각각 동일하면 구조적으로 동일하다고 봅니다.
동일하면 `1`, 아니면 `0`을 반환합니다.

함수 원형:
int identical(BTNode *tree1, BTNode *tree2);

예:
tree1 = `(1, 3, 2, 5, 4, 7, 8)`
tree2 = `(1, 3, 2, 5, 4, 7, 8)`
위 두 트리는 구조적으로 동일합니다.

샘플 메뉴:
1: tree1 생성
2: tree2 생성
3: 두 트리가 구조적으로 동일한지 확인
0: 종료

샘플 결과:
The resulting tree1 is: 1 3 2 5 4 7 8
The resulting tree2 is: 1 3 2 5 4 7 8
Both trees are structurally identical.

2. (maxHeight)
이진 트리의 루트 노드를 받아, 루트에서 가장 먼 리프 노드까지의 가장 긴 경로에 있는 링크 수를 반환하는 C 함수 `maxHeight()`를 작성하시오.
어떤 노드의 높이는 그 노드에서 가장 깊은 리프까지의 링크 수입니다.
빈 트리의 높이는 `-1`로 간주합니다.

함수 원형:
int maxHeight(BTNode *root);

예:
이진 트리 `(1, 2, 3, 4, 5, 6, 7)`의 최대 높이는 `2`입니다.

샘플 메뉴:
1: 이진 트리 생성
2: 최대 높이 찾기
0: 종료

샘플 결과:
The resulting binary tree is: 1 2 3 4 5 6 7
The maximum height of the binary tree is: 2

3. (countOneChildNodes)
이진 트리의 루트 노드를 받아 자식이 정확히 하나인 노드의 개수를 반환하는 C 함수 `countOneChildNodes()`를 작성하시오.

함수 원형:
int countOneChildNodes(BTNode *root);

예:
이진 트리 `(10, 20, 55, 30, 50, 60, 80)`에서 자식이 정확히 하나인 노드는 2개입니다.

샘플 메뉴:
1: 이진 트리 생성
2: 자식이 정확히 하나인 노드 수 세기
0: 종료

샘플 결과:
The resulting binary tree is: 10 20 55 30 50 60 80
The Number of nodes that have exactly one child node is: 2

4. (sumOfOddNodes)
정수로 이루어진 이진 트리의 루트 노드를 받아, 트리에 있는 모든 홀수 값의 합을 반환하는 재귀 C 함수 `sumOfOddNodes()`를 작성하시오.

함수 원형:
int sumOfOddNodes(BTNode *root);

예:
이진 트리 `(11, 40, 35, 50, 80, 60, 85)`에서 홀수 값의 합은 `131`입니다.

샘플 메뉴:
1: 이진 트리 생성
2: 트리의 모든 홀수 합 구하기
0: 종료

샘플 결과:
The resulting binary tree is: 11 40 35 50 80 60 85
The sum of all odd numbers in the binary tree is: 131.

5. (mirrorTree)
주어진 이진 트리를 거울상 구조가 되도록 수정하는 재귀 C 함수 `mirrorTree()`를 작성하시오.
중간 트리나 임시 트리를 새로 만들어서는 안 됩니다.
함수는 거울 변환할 이진 트리 루트 노드 포인터 하나를 매개변수로 받습니다.

함수 원형:
void mirrorTree(BTNode *node);

예:
원래 트리 `tree1 = (5, 6, 4, 3, 2, 1)` 이면,
거울 트리 `tree2 = (1, 2, 3, 4, 6, 5)`가 됩니다.

샘플 메뉴:
1: 이진 트리 생성
2: 이진 트리 거울 변환
0: 종료

샘플 결과:
The resulting binary tree is: 5 6 4 3 2 1
Mirror binary tree is: 1 2 3 4 6 5

6. (printSmallerValues)
이진 트리의 루트 노드와 정수 `m`을 받아, 트리에 저장된 값들 중 `m`보다 작은 모든 정수를 출력하는 C 함수 `printSmallerValues()`를 작성하시오.

함수 원형:
void printSmallerValues(BTNode *node, int m);

예:
이진 트리 `(25, 30, 65, 50, 10, 60, 75)`에 대해 `m = 55`로 호출하면
`50 30 25 10` 이 출력됩니다.

샘플 메뉴:
1: 이진 트리 생성
2: 주어진 값보다 작은 값 출력
0: 종료

샘플 결과:
The resulting binary tree is: 25 30 65 50 10 60 75
Enter an integer value to print smaller values: 55
The values smaller than 55 are: 50 30 25 10

7. (smallestValue)
주어진 트리에 저장된 값들 중 가장 작은 값을 반환하는 C 함수 `smallestValue()`를 작성하시오.
함수는 트리의 루트 노드 포인터를 매개변수로 받습니다.

함수 원형:
int smallestValue(BTNode *node);

예:
이진 트리 `(25, 30, 65, 50, 10, 60, 75)`의 최솟값은 `10`입니다.

샘플 메뉴:
1: 이진 트리 생성
2: 최솟값 찾기
0: 종료

샘플 결과:
The resulting binary tree is: 25 30 65 50 10 60 75
Smallest value of the binary tree is: 10

8. (hasGreatGrandchild)
적어도 한 명의 증손자(great-grandchild)를 가지는 모든 노드의 값을 출력하는 재귀 C 함수 `hasGreatGrandchild()`를 작성하시오.
함수는 이진 트리의 루트 노드 포인터 하나를 매개변수로 받습니다.

함수 원형:
int hasGreatGrandchild(BTNode *node);

예:
이진 트리 `(25, 30, 20, 65, 50, 10, 60, 75, 15)`에서 노드 `50`은 증손자를 가집니다.

샘플 메뉴:
1: 이진 트리 생성
2: 증손자를 가지는 노드 찾기
0: 종료

샘플 결과:
The resulting binary tree is: 25 30 20 65 50 10 60 75 15
The values stored in all nodes of the tree that has at least one great-grandchild are: 50
